PARTE I : LA WEB COMO UN LIENZO DE IDEAS

¿ Por qué realizar este curso ?
La web ha cambiado al mundo, y lo sigue haciendo, con personas como tú plasmando sus ideas en el lienzo de la web ¿Por qué no ser parte de este cambio?

La web, hoy en día, se ha convertido en todo un medio para el emprendimiento, y al mismo tiempo es uno de los campos laborales con más demanda en el ambiente tecnológico mundial.

En Uppersky disfrutamos de crear para la web y queremos compartir nuestras experiencias a través de una serie de cursos que hemos preparado. Si no conoces absolutamente nada sobre desarrollo web, esta primera parte es para ti.

Aclaramos que este curso no es para ayudarte en la universidad, es para qué aprendas a desarrollar tus ideas en la web, para que conozcas de que forma personas en todo el mundo están creando para la web y para que entres en este campo laboral de la mejor manera.

¿Cómo funciona la web?

Antes de crear para la web conoceremos cada uno de los componentes que la hacen posible. Exploraremos cada uno de estos conceptos desde su concepción original como una red de documentos, hasta su evolución en la plataforma tecnológica que es hoy en día.

¿Cómo se crea en la web?

Múltiples tecnologias son usadas para crear sitios, tantas que llegan a confundir, por ello nos dedicaremos a clasificar estas tecnologias (backend y frontend) y así comprender la finalidad de cada una.

Conociendo HTML

Tomaremos contacto del lenguaje base de la web, el cual nos permite indicar que es lo que tiene que ir en nuestra página web. Revisaremos las etiquetas básicas y las nuevas etiquetas semánticas de HTML, además de, sitios de documentación donde consultar más sobre HTML.

Conociendo CSS

Un sitio web necesita estilo, imagen, verse bien, darse una presencia, necesita formas, colores, efectos especiales si son necesarios, y CSS es el lenguaje de la web hecho para ello. Conoceremos los selectores y propiedades básicas de CSS para la maquetación de sitios. También revisaremos aspectos como los tipos de medidas, además de, detallar el modelo de caja, uno de los temas que mas pueden dar dolor de cabeza al momento de hacer un diseño web pero que es mas fácil de lo que parece.

El flujo de trabajo

Al momento de realizar cualquier desarrollo es bueno seguir cierto flujo de trabajo, cumplir con ciertas etapas y colocar algunas reglas que nos harán posible agilizar nuestros trabajo y poder trabajar con otros. Definir un flujo de trabajo (workflow) es importante y es lo que haremos en esta sección.

Maquetando un sitio famoso

Pondremos en práctica lo aprendido tomando una página de un sitio web conocido (Twitter, Facebook, YouTube, Google, etc) y realizaremos su maquetación base, definiendo las secciones que las componen, dando formas y refinando poco a poco cada sección hasta que el tiempo se nos agote.

CSS aún más cool

CSS3 en estos últimos años se ha vuelto extremadamente potente para el diseño web, y ha sido uno de los pilares para la expansión de las tecnologías web en otros ambientes como ha sido en la creación de aplicaciones móviles. Complementaremos lo antes aprendido de CSS con nuevas propiedades, pseudoclases, webfonts, las media queries, íconos basados en fuentes y una revisión básica de animaciones con CSS.

Refinando nuestro sitio

Retomaremos nuestro sitio web, nos dedicaremos a refinar una de las secciones con todos los elementos del sitio original. Aprenderemos sobre prácticas de diseño recomendadas al momento de hacer una página web como: proporciones, definición de colores, seleción de fuentes, uso correcto de los bordes, consejos en el uso de imágenes y otros.

La web más allá de este curso

Como veremos al inicio del curso, la web es toda una plataforma tecnológica, tan grande que no es abarcable en este curso. Por ello, esta última sección corresponderá a una charla sobre esos otros temas que debes aprender y que enseñaremos en el resto de esta serie de cursos. Hablaremos de temas como: javascript, el estandar HTML5, frameworks CSS, preprocesadores, backend, APIs, hostings, dominios y optimización en buscadores, además de responder a todas tus preguntas.
